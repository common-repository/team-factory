== Description ==

Lanpa Custom Contents plugin works only in Lanpa WordPress Theme. You can not use it for others. This plugin helps you to build a original Landing Page by adding and editting required sections. For example, You can add and edit "Solution", "Reason" and "Testimonials" sections with this plugin, which will be necessary for your Landing page. Enjoy editting texts, colors and iamges as you like.
== Installation ==

1. Upload `lanpa-custom-contents-plugin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
